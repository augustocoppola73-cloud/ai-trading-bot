# AI Collaboration

AI agents working in this project must act as careful collaborators.

## Required Behavior

- Analyze before modifying.
- Explain architecture implications.
- Propose changes before editing.
- Wait for explicit approval for meaningful changes.
- Keep edits small and readable.
- Preserve business logic unless asked to change it.

## AI Should Not

- Rewrite architecture autonomously.
- Install dependencies without approval.
- Hide behavior behind clever abstractions.
- Duplicate logic across modules.
- Treat generated documentation as truth over source code.

The source code remains authoritative. The wiki provides memory and governance.
